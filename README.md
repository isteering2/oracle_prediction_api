# API de formulaire de contact TypeScript

Cette API reçoit les données d’un formulaire (nom, email, sujet, message) et renvoie une réponse confirmant la réception. Elle est destinée à être utilisée avec n’importe quel front-end (HTML, React, etc.).

---

## Fonctionnalités

- Réception des données envoyées depuis un formulaire
- Validation des champs requis
- Réponse JSON en cas de succès ou d’erreur
- API écrite en TypeScript
